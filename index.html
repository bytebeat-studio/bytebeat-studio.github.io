ByteBeat Studio — Help

Transport

▶ Play / ■ Stop
Starts or stops audio. Space also toggles play/stop (when not typing in a text field).
↶ Undo / ↷ Redo
Steps backward/forward through your edit history (note placement, moves, resizes, deletes, cut/paste, track changes). Ctrl+Z / Ctrl+Y.
BPM
Tempo in beats per minute. Changes how many samples make up one beat, which shifts note timing in the generated formula.
LOOP BARS
How many bars the whole pattern lasts before it repeats seamlessly. The exported formula wraps time with t % (loop length in samples) so it plays forever without a hard stop or silence gap, just like classic bytebeat loops.

Piano Roll

Place a note
Click empty grid space. Uses the pitch you clicked and the current grid snap as the default length.
Move a note
Click and drag its body.
Resize a note
Drag its right edge (cursor changes to ↔ near the edge).
Delete a note
Right-click it, or select it and press Delete, or use the ✂ Cut tool and click it.
Multi-select
Shift+drag on empty space draws a selection rectangle. Shift+click adds/removes single notes from the selection.
Copy / Cut / Paste / Duplicate
Ctrl+C, Ctrl+X, Ctrl+V (pastes after your current selection), Ctrl+D duplicates immediately after the selected block.
Zoom
H = horizontal (time) zoom, V = vertical (pitch row height) zoom, using the +/− buttons in the toolbar.
Grid snap
Controls how finely notes snap to the timeline: 1/4, 1/8, 1/16, or off for free placement.

Sound vs. Notes (important distinction)

Sound (roll toolbar)
Chooses the timbre — the waveform shape (Classic Square, Saw, Bit Noise, Organ, and others) that the active track's notes are built from. This is a per-track setting.
Notes tab
Chooses the pitch that gets previewed and highlighted for reference — pick a note name and octave, or click the mini-keyboard to preview it. In the roll itself, the pitch of a new note is always whatever row you click; the Notes tab is for auditioning pitches and understanding the note-name/octave layout before you click.
Per-note Sound (double-click, or double-tap Shift)
Double-click a note in the piano roll to open a small menu right next to it. If you'd rather use the keyboard: select one note, then press Shift twice quickly (within about half a second) — both do the same thing. This menu uses its own descriptive names (Flat Beep, Buzzy Beep, Metallic Ring, and so on) for the same 9 timbres as the toolbar's Sound dropdown — the toolbar shows the technical names, this menu describes how each one sounds. Pick a sound and only that note switches to it — every other note on the track keeps using the track's regular Sound. Choose "Use track default" to remove the override and go back to following the track's Sound setting. Click outside the menu or press Esc to close it without changing anything.

Tracks

Track select / + Add track
Each track has its own notes, sound (timbre), and volume, and contributes its own term to the final summed formula.
M / S
Mute or Solo a track. If any track is soloed, only soloed tracks play; muted tracks never play.
Volume slider
0–100%, scales that track's contribution in the exported formula.
⧉ Duplicate / ✕ Delete
Copies a track including all its notes, or removes it (at least one track always remains).

Formula Generator

What it shows
The single classic-bytebeat expression that reproduces everything you've drawn: only t, integers, and + - * / % & | ^ ~ << >> — no custom functions. It pastes directly into any standard bytebeat composer.
classic-bytebeat OK badge
Confirms the current formula only uses the allowed operator set.
Copy formula / Export .txt
Copies the expression to your clipboard, or saves it as a plain-text file. This is just the final formula — good for pasting into another bytebeat player, but it can't be edited back into notes.
Export save / Import save
Saves your whole project — every track, note, tempo, and per-note sound override — as a downloadable file, unlike Export .txt which only saves the final formula. Export save to get a file you can send to another device; open ByteBeat Studio there and use Import save to pick that file and continue exactly where you left off. Import save replaces the current project. Ctrl+Z right after an accidental import restores your tracks and notes, but tempo/grid/loop settings stay at whatever the imported file had — double-check those after undoing.
